# Shell注释

## 单行注释

以 **#** 开头的行就是注释

```shell
#--------------------------------------------
# 这是一个注释
# author：litibai
# age: 18
#--------------------------------------------
##### 用户配置区 开始 #####
#
#
# 这里可以添加脚本描述信息
#
#
##### 用户配置区 结束  #####
```

## 多行注释

- 格式一: 

  ```shell
  :<<EOF
  注释内容...
  注释内容...
  注释内容...
  EOF
  ```

- 格式二: 

  ```shell
  :<<'
  注释内容...
  注释内容...
  注释内容...
  '
  ```

- 格式三: 

  ```shell
  :<<!
  注释内容...
  注释内容...
  注释内容...
  !
  ```
